Sobre Nosotros

Info sobre nuestro equipo

Nosotros somos un equipo de estudiantes del curso de Python dictado por CoderHouse. Pertenecemos a la comisión nro 34365, nuestro profesor es Daniel Ochoa y nuestro tutor es Pablo David Schvager. Debajo podrán leer una breve info de cada uno de los participantes de este grupo.

El objetivo de este trabajo es crear una web con todas las funcionalidades aprendidas en clase, presentándolas en una entrega final para aprobar el curso. De forma dinámica trabajamos en grupo acordando las tareas a realizar cada uno para llevar adelante el blog. También nos ayudamos para resolver los problemas que lleva entregar un trabajo final.
